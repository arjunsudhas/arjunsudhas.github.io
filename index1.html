EYECLOUD: AN EVOLUTION TO MEDICAL RECORD SYSTEM.

Posted In: Application devolopment

25 175


Eye Cloud is a database based application which is used to improvise the medical field by making a track record of a patient’s treatment histories. Eye Cloud uses biometric identification techniques to identify a patient thereby eliminates the necessity of a registration card. A conventional file record need not be kept for each person since our application provides a cloud data base which can be accessed from anywhere around the globe. Treatment records are being collected at birth and each hospital visits,treatments undergone, consulted doctors, prescribed medicines and much more information are collected. Doctors will find it easy to prescribe medicines or to advice a treatment since they are given with entire medical history of the patient. Eye Cloud mainly uses the Retinal Scanning technology to store unique patterns on a person's retinal blood vessels which can be used to identify a patient uniquely. The record collection procedure starts from the birth in which the initial birth conditions (like weight, disorders etc.) are collected. Patients are provided with an android application in which they can track their medical histories, view consulted doctors and book appointments etc. They can also view their prescribed medicines from various doctors consulted over the past.

CAB4U:CAB BOOKING DEKTOP APP - Mini Project.

Posted In: Java App

57 101


The project primarily emphasize on a taxi booking service which is customer friendly and user friendly. The software primarily focuses on the activities of a taxi service firm such as booking a cab, checking availability of drivers/cars in the firm, updating the current status or changing them accordingly for drivers and vehicles, calculating distance or finding routes. All details are securely stored in the database which provides all advantages of a database such as durability, security, reusability etc. The project has all the provisions to automate the allocation of available drivers, cabs for a particular customer, store the details of customer and vehicle in the database, to generate booking, summary for each cab booking, to fetch previous transaction details. The system is user friendly and secured with user privileges. The above factors made our system more efficient and powerful.
